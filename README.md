# Olist E-commerce Data Analysis

## Project Overview
This project analyzes e-commerce data to extract business insights related to customer behavior, shipping performance, and payment methods.

---

## Dataset
- 117,000+ orders  
- Multiple related tables (orders, payments, reviews, products, sellers)

---

## Key Insights
- Shipping delays increase negative reviews by 2x  
- Credit card users spend 35% more than others  
- Top 5% of customers generate 30% of total revenue  

---

## Tools Used
- Python (Pandas, Seaborn)  
- SQL Server  
- Power BI  
- Excel  

---

## Business Recommendations
- Improve shipping efficiency in high-delay regions  
- Introduce loyalty programs for top customers  
- Promote high-value payment methods
